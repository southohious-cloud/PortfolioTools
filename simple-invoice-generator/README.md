# Simple Invoice Generator

A lightweight utility that creates clean, readable invoices from a simple JSON input. Designed for quick billing, client summaries, and lightweight workflow automation.

---

## Purpose

Generates consistent, professional invoices without templates, spreadsheets, or manual formatting.

---

## Features

- JSON-based input for client details and line items
- Automatic subtotal and total calculation
- Clean, plain-text invoice output
- Consistent formatting for archiving or email
- Works as a standalone tool or part of a workflow

---

## How to Use

Run the script:
python invoice_generator.py

Enter the path to a JSON file containing invoice details.

A formatted invoice will be printed to the terminal.
