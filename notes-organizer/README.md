# My Notes Organizer

A simple tool that turns raw notes into a clean, structured summary.

## What It Does
- Extracts bullet points
- Extracts action items
- Groups remaining notes into a clean "Additional Notes" section
- Saves the result to `organized_notes.txt`

## How It Works
1. Paste raw notes line-by-line.
2. The tool automatically detects:
   - Bullet points
   - Action items (todo, follow up, next step)
   - Miscellaneous notes
3. Outputs a clean, organized summary.

## Output Example
Key Points
- Example bullet

Action Items
- Follow up with client

Additional Notes
- Miscellaneous detail
